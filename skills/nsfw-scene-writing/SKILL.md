# 情欲场景写作

## 概述

专门处理小说中成人内容（H场景/肉戏）的写作。提供场景分级、模板选择、节奏控制、感官描写指南，以及安全词/敏感词管理。确保情欲场景既能满足读者的期待，又保持文学性和角色一致性。

## 前置条件

故事项目必须已存在，至少包含：
- `story.md`（含NSFW设定：分级、尺度、边界）
- 至少1个角色（用于确定场景中的角色动态）
- 场景所涉及的角色文件（了解其性向、偏好、禁忌和身体特征）

## 场景分级体系

将情欲场景按进度和露骨程度分级：

| 级别 | 名称 | 描述 | 典型内容 |
|------|------|------|---------|
| L1 | 暧昧/暗示 | 眼神、触碰、气氛渲染 | 暧昧对话、不经意的身体接触、心跳加速 |
| L2 | 前戏/调情 | 接吻、爱抚、挑逗 | 深吻、抚摸、脱衣、口舌调情 |
| L3 | 边缘性行为 | 非插入式的亲密接触 | 手交、口交、腿交、乳交、摩擦 |
| L4 | 插入/结合 | 直接的性行为 | 各种体位的插入、双修神交 |
| L5 | 高潮/释放 | 高潮时刻的描写 | 身体反应、情感爆发、灵力/能量流动（仙侠） |
| L6 | 事后/温存 | 事后的互动与情感交流 | 拥抱、对话、清理、承诺 |

## 场景模板

根据场景的类型和氛围选择模板：

| 模板 | 氛围 | 角色动态 | 典型场景 |
|------|------|---------|---------|
| 初次 | 紧张、探索、温柔 | 引导者/被引导者 | 新婚夜、初夜、破处 |
| 温存 | 亲密、信任、放松 | 平等/互爱 | 日常恩爱、久别重逢 |
| 激情 | 急切、狂野、失控 | 双方主动 | 久别重逢、酒后、发情期 |

| 强迫 | 恐惧、屈辱、疼痛 | 加害者/受害者 | 强暴、胁迫（需注意写作伦理） |
| 偷窥 | 禁忌、刺激、隐秘 | 偷窥者/被偷窥者 | 暗中观察、自慰、他人交合 |

## 写作流程

### 1. 准备阶段

1. 阅读 `story.md` 确认该作品的描写尺度和边界
2. 阅读涉及角色的档案（性向、偏好、禁忌、身体特征）
3. 阅读上一章的结尾状态（情感状态、身体状态）
4. 阅读 `plot/arcs/` 了解此场景要推进的角色关系进展
5. 确认场景的叙事目的：纯粹的情欲满足？角色关系里程碑？剧情推进？

### 2. 场景设计

根据角色动态和场景模板设计场景：
- **场景级别**: 从哪一级开始，到哪一级结束？
- **情感铺垫**: 场景之前需要多少情感/剧情铺垫？
- **关键节拍**: 场景中的情绪转折点
- **角色变化**: 场景结束后角色的关系/心理/身体状态变化

### 3. 写作执行

按照 `references/nsfw-writing-guidelines.md` 中的写作指南执行：

**核心原则：**
- **情感先行**: 好的情欲场景建立在角色之间的情感张力之上。先写情感，再写身体
- **视角一致**: 严格保持在POV角色的感知范围内——ta所见、所闻、所触、所感、所想
- **虚实结合**: 既有具体的感官细节（"他的指尖沿着她的脊柱缓缓滑下"），也有适当的情感叙述和心理活动
- **节奏控制**: 
  - 前戏/铺垫用长句，细腻描述
  - 激情/高潮用短句，加快节奏
  - 事后用中长句，舒缓慢收
- **尊重禁忌**: 严格遵守角色设定中的禁忌和作品的边界声明
- **避免陈词滥调**: 避免过度使用"嗯啊"、"不要"、"好大"等俗套表达
- **类型适配**: 
  - 仙侠: 加入灵力流动、功法运转、境界感悟等元素
  - 都市: 注重服饰、场景、社会关系的真实感
  - 奇幻: 利用种族特性、魔法效果等世界观元素
  - 纯官能: 以感官描写为主，剧情为辅

### 4. 写后检查

完成场景写作后：
1. 检查角色行为是否符合其性格和偏好设定
2. 检查场景级别是否与作品的 `explicit-level` 一致
3. 检查是否触犯了任何宣言的禁忌
4. 检查感官描写的多样性（触觉、视觉、听觉、嗅觉、味觉的运用）
5. 更新连续性状态：角色关系进展、身体状态变化、好感度变化

## 敏感词/安全词管理

参照 `references/sensitive-words.md` 中的敏感词词典。在写作时：
- 使用文学化的表达替代直白的器官描述
- 不同尺度的作品使用不同程度的隐晦/露骨词汇
- 允许用户在 `story.md` 中自定义替换词典
